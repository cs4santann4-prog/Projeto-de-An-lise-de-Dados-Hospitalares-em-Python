# Análise de Dados Hospitalares em Python

## Descrição do Projeto
Projeto de análise de dados desenvolvido para a disciplina de **Tópicos de Big Data em Python**, com o objetivo de transformar dados hospitalares brutos em insights visuais estratégicos para apoio à gestão em saúde.

## Objetivos
- Desenvolver sistema de análise de dados para serviços de saúde
- Implementar visualizações gráficas de indicadores hospitalares  
- Criar metodologia de análise contínua de desempenho
- Oferecer insights baseados em dados para tomada de decisão

## Tecnologias Utilizadas
- **Python 3.x**
- **Pandas** - Manipulação e análise de dados
- **Matplotlib** - Visualizações gráficas
- **Jupyter Notebook/Google Colab** - Ambiente de desenvolvimento

## Como Executar

### Pré-requisitos
- Python 3.7 ou superior
- Bibliotecas: pandas, matplotlib

### Instalação das Dependências
```bash
pip install pandas matplotlib
```

### Execução do Projeto
```bash
python analise_dados_hospitalares.py
```

## Funcionalidades
-  Análise de distribuição de serviços hospitalares
-  Gráfico de pizza - Percentual por serviço
-  Histograma - Idade dos pacientes  
-  Gráfico de satisfação por serviço
-  Dashboard integrado com subplot_mosaic

## Resultados Esperados
- Distribuição percentual dos serviços: Emergência, Cirurgia, Clínica Geral, UTI
- Índice de satisfação médio: 78.8 pontos
- Análise demográfica completa dos pacientes
- Visualizações profissionais para tomada de decisão
